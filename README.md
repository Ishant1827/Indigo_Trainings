# Indigo Trainings & IT Consultancy Services

A modern and responsive training institute website developed using HTML, CSS and JavaScript. The website provides complete information about courses, summer training programs, resources, demo classes, payment options and contact details.

## Features

- Responsive Design
- Modern UI/UX
- Summer Training Enrollment
- UPI Payment Integration
- QR Code Payment Support
- Demo Classes Section
- Training Resources
- Course Showcase
- Ambassador Program
- Team Section
- Contact & Location Section
- WhatsApp Integration
- Smooth Scrolling Navigation
- Loading Screen Animation
